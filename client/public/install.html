Installation Tips

Correct installation is one of the most important factors affecting long-term performance and appearance of any surface material. In this section, Comperra shares general knowledge and considerations for preparing, handling, and installing common flooring and wall products.

While installation techniques may vary depending on product type, job site conditions, or local code, we aim to offer clear foundational insights.

Topics include:

  • Substrate preparation and leveling
  • Moisture testing and mitigation
  • Underlayment and setting materials
  • Expansion joints and perimeter spacing
  • Correct orientation and layout
  • Grout and adhesive curing times
  • Environmental acclimation for wood and vinyl

Professional Guidance Notice

Our installation guidance is developed with input from trade professionals and technical manuals, but is not a substitute for manufacturer instructions or licensed contractor expertise. Always verify that your methods align with the most current product specifications and local building standards.

Tile Installation

Preparation Steps

  • Ensure substrate is level, clean, and structurally sound
  • Check for moisture issues and install barriers if needed
  • Plan layout to minimize cuts and maintain symmetry
  • Acclimate tiles to room temperature

Installation Tips

  • Use appropriate adhesive for tile type and substrate
  • Maintain consistent spacing with tile spacers
  • Check for level and lippage regularly
  • Clean excess adhesive from joints immediately

Grouting Guidelines

  • Wait 24-48 hours before grouting
  • Choose grout color that complements tiles
  • Remove haze promptly with clean water
  • Allow proper curing time before sealing

Stone & Slab Installation

Pre-Installation

  • Verify measurements and template accuracy
  • Inspect slabs for defects or color variations
  • Ensure adequate structural support
  • Plan for expansion joints if required

Installation Process

  • Use proper lifting equipment for heavy slabs
  • Apply adhesive according to manufacturer specs
  • Check level and adjust as needed
  • Seal joints with appropriate sealant

Finishing Work

  • Polish edges to desired finish
  • Apply sealer if required for stone type
  • Install edge treatments or backsplashes
  • Final cleanup and inspection

LVT Installation

Subfloor Preparation

  • Ensure subfloor is level within 3/16" over 10 feet
  • Remove all debris and vacuum thoroughly
  • Check moisture levels and install barriers if needed
  • Allow LVT to acclimate per manufacturer instructions

Installation Methods

  • Click Lock: Floating installation with tongue and groove
  • Glue Down: Permanent adhesive installation
  • Loose Lay: Friction-fit system for easy replacement

Best Practices

  • Stagger joints for natural appearance
  • Leave expansion gaps around perimeter
  • Use appropriate transition strips
  • Roll with 100lb roller for glue-down installations

Hardwood Installation

Acclimation Process

  • Deliver wood to job site 3-5 days before installation
  • Maintain 60-80°F temperature and 30-50% humidity
  • Stack with spacers for air circulation
  • Check moisture content before installation

Installation Options

  • Nail Down: Traditional method for solid wood
  • Glue Down: For engineered wood on concrete
  • Floating: Click-lock system over underlayment

Finishing Steps

  • Sand and finish site-finished floors
  • Install transition strips and moldings
  • Apply final protective coating if required
  • Maintain proper humidity during curing

Radiant Heating Installation

Planning Phase

  • Calculate heat loss and system requirements
  • Plan cable/mat layout avoiding fixtures
  • Verify electrical capacity and install GFCI
  • Test resistance before and after installation

Installation Process

  • Install insulation and thermal breaks
  • Secure heating elements per layout plan
  • Connect to thermostat and test system
  • Apply leveling compound over cables

Safety Considerations

  • Use only qualified electricians for connections
  • Install appropriate breakers and GFCI protection
  • Test system before covering with flooring
  • Provide documentation for future reference

Carpet Installation

Preparation Requirements

  • Clean and level subfloor thoroughly
  • Install appropriate padding or underlayment
  • Acclimate carpet in installation environment
  • Plan seam placement for best appearance

Installation Techniques

  • Stretch-in: Traditional tack strip method
  • Glue Down: Direct adhesive installation
  • Double Stick: Pad adhered, then carpet

Quality Control

  • Ensure proper stretch to prevent wrinkles
  • Check seam alignment and security
  • Vacuum thoroughly before furniture placement
  • Install transition strips at doorways
